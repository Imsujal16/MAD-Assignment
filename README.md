# MAD Assignment Projects

This workspace contains four separate Android Studio projects written in Java, one for each assignment question.

## Project Folders

- `Q1_CurrencyConverter`
  - Currency converter for INR, USD, JPY, and EUR
  - Includes a settings screen to toggle light/dark theme

- `Q2_MediaPlayer`
  - Plays a local audio file from device storage
  - Streams a video from a URL
  - Includes Play, Pause, Stop, and Restart controls

- `Q3_SensorReader`
  - Reads and displays Accelerometer, Light, and Proximity sensor values

- `Q4_CameraGallery`
  - Lets the user choose a folder
  - Captures photos into that folder
  - Shows images in a grid gallery
  - Opens image details and supports deleting an image with confirmation
